Une communauté indigène produit des pantoufles en jute et a créé un site web pour vendre sa production en ligne. Les pantoufles sont d'un seul type, mais elles sont produites en différentes tailles. Le stock peut être vu comme un tableau avec une seule ligne, dans lequel chaque colonne (indice de base 1) représente une taille. Lorsqu'un chausson est vendu, le stock doit être mis à jour. Si le stock pour une taille de pantoufles a une valeur nulle, les pantoufles de cette taille ne peuvent pas être vendues. En d'autres termes, la vente n'a pas lieu.

Étant donné le stock initial et la liste des commandes des clients, écrivez un programme permettant de déterminer le nombre total de pantoufles vendues. Chaque commande concerne une seule pantoufle. Les ventes sont traitées séquentiellement, dans l'ordre dans lequel les commandes ont été passées. Si une vente n'est pas possible en raison d'un manque de stock, la commande correspondante est ignorée.
